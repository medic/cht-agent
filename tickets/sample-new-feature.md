title: Allow bulk contact creation using CSV upload
type: feature
priority: medium
domain: contacts
---

## Description
Creating contacts one by one is time consuming, especially when onboarding large numbers of health workers or facilities during deployments.

## Expected Behavior
Users should be able to upload a CSV file and create multiple contacts in one go.

## Acceptance Criteria
- [ ] CSV upload option available in webapp
- [ ] Basic validation for required fields
- [ ] Invalid rows are reported clearly
- [ ] Valid entries are created successfully
